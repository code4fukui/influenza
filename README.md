# influenza

This project visualizes the number of reported influenza patients in Japan over the years.

## Demo
- https://code4fukui.github.io/influenza/
- https://code4fukui.github.io/influenza/year.html

## Features
- Displays the number of reported influenza patients in Japan over the years
- Provides yearly and weekly visualizations

## Data / API
The data is from the open data source:
- [Influenza-related press release materials | Ministry of Health, Labour and Welfare](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/kekkaku-kansenshou01/houdou.html)
