# インフルエンザ患者報告数の可視化

本リポジトリは、日本の季節性インフルエンザの患者報告数を可視化するためのウェブアプリケーションです。厚生労働省のオープンデータを使用して、週単位および年単位での推移グラフを表示しています。

## デモ
- https://code4fukui.github.io/influenza/
- https://code4fukui.github.io/influenza/year.html

## 機能
- 季節性インフルエンザの患者報告数の週単位の推移グラフ
- 患者報告数の年単位の推移グラフ

## データ・API
本アプリケーションでは、厚生労働省のオープンデータ「[インフルエンザに関する報道発表資料](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/kekkaku-kansenshou01/houdou.html)」を使用しています。
