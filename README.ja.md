# influenza

日本で報告された季節性インフルエンザ患者数を可視化するウェブアプリケーションです。厚生労働省が提供するオープンデータを活用し、インタラクティブなチャートを生成します。

## デモ

このプロジェクトでは、インフルエンザデータを2つの異なる形式で可視化します:

1.  **連続タイムラインビュー:** 取得可能な全期間にわたる週ごとのインフルエンザ報告数を、1つの連続したチャートで表示します。
    -   https://code4fukui.github.io/influenza/

    
![連続タイムラインビューのスクリーンショット。複数年のインフルエンザ症例数を示す単一の折れ線グラフが表示されています。](https://user-images.githubusercontent.com/1595839/213852899-8884976f-319a-41d1-9257-2796113800e8.png)


2.  **年次比較ビュー:** 複数年のデータを週ごとの軸に重ね合わせて表示し、季節ごとの傾向を比較します。
    -   https://code4fukui.github.io/influenza/year.html

    
![年次比較ビューのスクリーンショット。1つのグラフに複数の色の線が表示され、それぞれが異なる年のインフルエンザ傾向を表しています。](https://user-images.githubusercontent.com/1595839/213852906-89849226-c56a-4933-9118-293677b1029c.png)


## 機能

-   **連続タイムライン:** 複数年にわたる週ごとの患者報告数を時系列で追跡します。
-   **年次比較:** 1つのチャートに年ごとのデータを重ね合わせ、インフルエンザのシーズンごとの傾向を比較します。
-   **インタラクティブなチャート:** データポイントにカーソルを合わせると、その週の具体的な報告数を確認できます。
-   **軽量:** サーバーサイドの依存関係がなく、すべてブラウザ上で動作します。

## 使い方 / 開発

本プロジェクトは静的なウェブアプリケーションであり、ウェブサーバーなしでローカル環境で実行できます。

1.  リポジトリをクローンします:
    ```bash
    git clone https://github.com/code4fukui/influenza.git
    ```
2.  プロジェクトディレクトリに移動します:
    ```bash
    cd influenza
    ```
3.  `index.html` または `year.html` をウェブブラウザで直接開きます。

## データソース

データは、日本政府が提供する以下のオープンデータを使用しています:

-   [インフルエンザに関する報道発表資料 ｜厚生労働省](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/kekkaku-kansenshou01/houdou.html)
